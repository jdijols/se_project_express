# WTWR (What to Wear?): Back End

## About

The back-end project is focused on creating a server for the WTWR application. Users can sign up, log in, create clothing items for different weather conditions, and like/unlike items. The server handles user authentication with JWT tokens and protects routes so only authorized users can modify data.

## Technologies and Techniques

- Node.js and Express.js for the server
- MongoDB and Mongoose for the database
- JWT (JSON Web Tokens) for authentication
- bcryptjs for password hashing
- validator for email and URL validation
- RESTful API design
- Centralized error handling with status codes

## Running the Project

`npm run start` — to launch the server

`npm run dev` — to launch the server with the hot reload feature

`npm run lint` — to run the linter
